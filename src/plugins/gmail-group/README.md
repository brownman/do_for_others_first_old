plugin: gmail-group
=
how to use:
---
- run: ./breath.sh gmail-group
- open file: vars.cfg + edit: set your password and username
- edit: gmail-notify configuration at: ~/.notifier.conf (optional: use my settings: at:./BAK/notifier.conf )
